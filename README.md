# T-Gate Dropping for Transversal Code Switching

Code accompanying the MMathPhys dissertation: 
*Towards Fault-Tolerant Universal Quantum Computation: Transversal Code Switching*, 
University of Oxford, Trinity 2026.

## Contents
- `T-dropping_code.ipynb` — Jupyter notebook containing all analyses of Chapter 4, including the diamond-norm error model, min-cut switching solver, and Pareto frontier enumeration.

## Dependencies
- `numpy`
- `networkx`
- `qutip` (for two-qubit diamond norm evaluation via SDP)
- `matplotlib` (plotting)
